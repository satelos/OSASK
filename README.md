# OSASK
- 是一本操作系统制作的书籍, 本项目将从头开始实践这本书
- 最终将提供一个可方便运行的版本
- 作者网站 http://hrb.osask.jp/
- 系统的名字叫`HariboteOS`或者说`はりぼてOS`

## 工具链说明

原作者是在 `Windows XP` 环境下开发的, 并且貌似没给`nask`提供 `*nix` 平台可用的汇编编译程序

本项目将在汇编部分将把语法转换为比较通用的`nasm`, 这样应该全平台可编译
