# OSASK
- OSASK 是 《30天自制操作系统》 内的系统名称, 本项目将从头开始实践这本书
- 改造使其额外支持 MacOS、Linux、nasm
- 最终将提供一个可运行的版本, 并提供 docker 镜像及系统镜像两种方式来运行

## 工具链说明
原作者是在 Windows 环境下开发的, 并且貌似没给"nask"提供 OSX 可用的汇编编译程序

本项目将在 Mac 环境下开发, 汇编部分将把语法转换为比较通用的 nasm, 这样应该全平台可用
